octave number

Octave is an integer value from -5 to 5 inclusive.
accidental string

Accidental is either of

  • '#' - sharp
  • '##' - double sharp
  • 'b' - flat
  • 'bb' - double flat
  • 'n' - natural
  • '' - (none)
