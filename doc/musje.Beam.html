Class: Beam

musje. Beam

new Beam(value, level, parent)

A beam is a horizontal or diagonal line used to connect multiple consecutive notes (and occasionally rests) in order to indicate rhythmic grouping. Only eighth notes (quavers) or shorter can be beamed.

Beam is created by musje.Cell#makeBeams and attached to musje.Durable in {@link musje.Durable#beams}[level]

Parameters:
Name Type Description
value string

Beam value: 'begin', 'continue' or 'end'.
level number

Beam level starting from 0 to up.
parent musje.Durable

The parent durable music data.
